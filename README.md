# üé® Artistome - Art Selling Platform

**Artistome** is a basic art-selling platform developed from scratch to facilitate the buying and selling of art. The project emphasizes simplicity and functionality, focusing on core features like secure payments and artwork listing.

---

## üåü Features

- **User Authentication**: Secure login and registration system for both artists and buyers.
- **Art Categories**: Explore art by categories like paintings, sculptures, photography, and more.
- **Art Listings**: Detailed information about each piece, including dimensions, medium, and pricing.
- **Search and Filters**: Intuitive search and filtering options to find specific artwork easily.
- **Cart**: Add Art to you cart.
- **Secure Payment Gateway**: Integrated for smooth and safe transactions.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## üöÄ Installation & Setup

Follow these steps to set up the project locally:

### Prerequisites
- Node.js installed
- MongoDB installed and running

### Clone the Repository
```bash
git clone https://github.com/your-username/artistome.git
cd artistome
```
### Backend Setup
1. Navigate to the project directory:
```bash
cd backend
```
2. Install dependencies:
```bash
npm install
```
3. Create a .env file and configure the following:
```env
PORT=5000
MONGO_URI=your-mongodb-connection-string
PAYMENT_API_KEY=your-payment-api-key
```
4. Start the backend server:
```bash
npm start
```
### Access the Application
- Backend: http://localhost:3000
